# UI DESIGN PRINCIPLES

## 1. REAL-TIME FEEL

Everything should:

- update live
- stream continuously
- animate smoothly
- feel responsive

Static dashboards lose hackathons.

---

## 2. MISSION-CONTROL AESTHETIC

UI inspiration should resemble:

- autonomous systems
- industrial robotics
- aerospace operations
- AI control centers

---

## 3. INFORMATION HIERARCHY

Critical events must dominate visually:

- anomalies
- mission failures
- trust drops
- blockchain confirmations

---

## 4. VISUAL STORYTELLING

Frontend should naturally tell the story:

mission → telemetry → proof → blockchain → trust evolution.

---

## 5. DEMO-FIRST DESIGN

Every screen should be:

- visually impressive
- understandable in seconds
- optimized for presentations
